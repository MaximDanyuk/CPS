
## О проекте

Веб страница реального сервиса ремонта техники

## Технологический стек и функциональность:

### HTML

- Семантические теги
- Работа с формами, инпутами

### CSS

- Flexbox
- Grid Layout
- Адаптивная верстка и отзывчивая верстка, использование медиа-запросов
- БЭМ Nested(Упрощенный)
- Базовые CSS технологии

### JavaScript

- Открытие модалок
- Использование классов ES6
- Webpack
- Использование библиотеки Swiper
